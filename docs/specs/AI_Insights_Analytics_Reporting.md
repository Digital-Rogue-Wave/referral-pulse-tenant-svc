## Optimization Strategies

1. **Use Haiku for simple tasks** (saves ~60%)
   - Anomaly detection scans
   - Simple classifications
   - Data formatting

2. **Cache common responses**
   - Industry benchmarks
   - Standard recommendations
   - Repeated queries

3. **Batch processing**
   - Generate insights during off-peak
   - Pre-compute daily summaries
   - Don't regenerate on every page load

4. **Smart triggers**
   - Only generate insights when data changes significantly
   - Throttle natural language queries per user
   - Weekly reports, not daily

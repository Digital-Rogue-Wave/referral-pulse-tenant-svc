1) IRSA
   Use the Terraform output:
   --set serviceAccount.annotations."eks\.amazonaws\.com/role-arn"=$(terraform output -raw irsa_role_arn_app)

2) External Secrets
   --set externalSecrets.enabled=true \
   --set externalSecrets.secretStoreRef.name=aws-secrets-manager \
   --set externalSecrets.db.remoteSecretName=$(terraform output -raw db_credentials_secret) \
   --set externalSecrets.redis.remoteSecretName=$(terraform output -raw redis_credentials_secret)

3) SQS (each with DLQ)
   --set queues.campaignUrl=$(terraform output -json sqs_urls | jq -r .campaign) \
   --set queues.campaignDlqUrl=$(terraform output -json sqs_urls | jq -r .campaign_dlq) \
   --set queues.invitationUrl=$(terraform output -json sqs_urls | jq -r .invitation) \
   --set queues.invitationDlqUrl=$(terraform output -json sqs_urls | jq -r .invitation_dlq) \
   --set queues.trackingEventsUrl=$(terraform output -json sqs_urls | jq -r .tracking_events) \
   --set queues.trackingEventsDlqUrl=$(terraform output -json sqs_urls | jq -r .tracking_events_dlq)

4) S3 (required)
   --set s3.mainBucket=$(terraform output -json s3_buckets | jq -r .main) \
   --set s3.staticBucket=$(terraform output -json s3_buckets | jq -r .static) \
   --set s3.backupsBucket=$(terraform output -json s3_buckets | jq -r .backups)

5) Ingress (ALB)
   Provide your ACM cert ARN and hostname in values (ingress.*). The AWS Load Balancer Controller will create/manage the ALB.

6) Probes & Downward API
   The Deployment sets POD_NAME/POD_NAMESPACE and readiness /readyz matches your leader-only behavior.
